greenShoes - Plataforma de E-commerce para Calçados Sustentáveis

Descrição do Projeto: O greenShoes é uma plataforma de e-commerce dedicada à venda de calçados sustentáveis. O projeto foi desenvolvido com o objetivo de oferecer uma experiência de compra online intuitiva e agradável, promovendo produtos que respeitam o meio ambiente.

Estrutura do Projeto:

.vscode: Configurações do ambiente de desenvolvimento.
Img.footer: Imagens utilizadas no rodapé do site.
css: Arquivos de estilo que definem a aparência e o layout do site.
img: Pasta contendo todas as imagens do site, incluindo fotos de produtos.
logo: Arquivos de logotipo da marca greenShoes.
carrinho.html: Página do carrinho de compras onde os usuários podem revisar e modificar seus itens selecionados.
carrinho.js: Script que gerencia as funcionalidades do carrinho de compras.
checkout2.html: Página de checkout onde os usuários finalizam suas compras.
home.html: Página inicial do site, apresentando os principais produtos e ofertas.
paginaDeCompra.html: Página de detalhes do produto, onde os usuários podem ver informações detalhadas e adicionar itens ao carrinho.
slider.js: Script que controla o carrossel de imagens na página inicial.
sobre.html: Página sobre a empresa, sua missão e valores.
Funcionalidades Principais:

Navegação Intuitiva: Interface amigável que facilita a navegação entre as diferentes seções do site.
Carrinho de Compras: Funcionalidade que permite aos usuários adicionar, remover e revisar produtos antes de finalizar a compra.
Checkout Seguro: Processo de checkout simplificado e seguro para garantir uma experiência de compra tranquila.
Slider de Imagens: Carrossel de imagens na página inicial para destacar produtos e promoções.
Informações Detalhadas dos Produtos: Páginas dedicadas a cada produto, com descrições detalhadas, imagens e opções de compra.
Objetivo: O objetivo do greenShoes é proporcionar uma plataforma de e-commerce eficiente e sustentável, promovendo a venda de calçados que respeitam o meio ambiente e atendem às necessidades dos consumidores conscientes.
